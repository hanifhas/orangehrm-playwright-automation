# OrangeHRM Automation Testing using Playwright

## Overview

This project demonstrates automated testing for login and logout functionality using Playwright.

## Test Coverage

- Login with valid credentials
- Login with wrong password
- Login with wrong username
- Login with empty fields
- Logout successfully

## Tools

- Playwright
- JavaScript
- Node.js

## Test Result

15 tests passed across:

- Chromium
- Firefox
- WebKit

## How to Run

npx playwright test
